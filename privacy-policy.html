UniTab 隐私政策

生效日期：2026年5月8日　|　最后更新：2026年5月9日

一、引言

UniTab（以下简称"本扩展"）是一款浏览器新标签页替换扩展，由个人开发者 Un1quer（以下简称"我们"）开发和维护。我们高度重视你的隐私和个人信息保护。

本隐私政策旨在向你说明本扩展如何处理你的信息，包括信息的收集、使用、存储和保护方式。请你仔细阅读本政策。安装或使用本扩展，即表示你同意本政策所述的信息处理方式。

二、我们收集什么信息

2.1 我们不收集的信息

本扩展不收集、不存储、不传输任何可识别个人身份的信息，包括但不限于：姓名、电子邮箱、电话号码、地理位置、IP 地址、设备标识符、浏览历史、书签数据。

2.2 本地存储的信息

以下数据仅存储在你设备的浏览器本地存储（localStorage）中，完全离线，不会上传至任何服务器：

  • 背景模式偏好（纯色 / 壁纸）
  • 自定义背景颜色值
  • 壁纸图片文件（本地压缩后以 Base64 格式存储）
  • 字体、圆角、阴影强度等界面偏好
  • 显示模式偏好（跟随系统 / 深色 / 浅色）
  • 当前选择的搜索引擎
  • 设置按钮位置偏好

你可以随时通过浏览器清除网站数据来删除上述所有本地信息。

三、网络请求

本扩展仅在以下场景发起网络请求，不附加任何用户标识信息：

3.1 搜索建议

当你在搜索框中输入关键词时，扩展会将该关键词发送到当前选择的搜索引擎（Google、百度、Bing 或 DuckDuckGo）以获取自动补全建议。这与你在浏览器地址栏或搜索引擎网站直接输入时的行为完全一致。该请求不包含任何个人身份信息或设备标识符。

3.2 网站图标

扩展可能请求 favicon.im 服务以获取搜索引擎对应的网站图标，该请求仅用于界面展示。

四、权限说明

本扩展在浏览器中声明了以下权限，均仅用于实现核心功能：

  • 存储（storage）：用于保存用户偏好设置的备用存储接口。
  • 新标签页覆盖（newtab）：用于替换浏览器默认新标签页，显示 UniTab 界面。
  • 主机权限（host_permissions）：用于向所选搜索引擎发送搜索建议请求，以及获取搜索引擎图标。仅限声明的域名。

五、第三方服务

  • 本扩展不集成任何第三方统计分析 SDK、广告联盟、用户画像或行为跟踪服务。
  • 搜索建议请求直接发往各搜索引擎的公开 API 接口，各搜索引擎的隐私政策请参见其官方网站。

六、信息安全

所有个性化设置和壁纸数据均存储在浏览器本地沙箱中，受浏览器自身的安全机制保护。由于数据完全不上传至任何外部服务器，不存在服务器端数据泄露的风险。

你应妥善保管自己的设备，避免他人物理接触导致本地数据被查看。

七、你的权利

根据《中华人民共和国个人信息保护法》及其他适用法律法规，你享有以下权利：

  • 知情权：了解我们如何处理你的信息（即本政策所述内容）。
  • 查阅与复制权：所有本地存储数据可通过浏览器开发者工具查看和导出。
  • 更正权：你可以在扩展设置面板中随时修改各项偏好。
  • 删除权：你可以通过清除浏览器数据、卸载扩展来彻底删除所有本地存储信息。
  • 撤回同意：你可以随时卸载本扩展以撤回对信息处理的同意。
  • 投诉举报：如你认为我们的信息处理行为侵犯了你的合法权益，可向履行个人信息保护职责的部门投诉举报。

八、儿童隐私保护

本扩展不针对 14 周岁以下的未成年人。我们不会有意收集未成年人的个人信息。如果你是监护人且发现被监护人未经同意使用了本扩展，请通过下方联系方式告知我们。

九、政策更新

我们可能根据法律法规变化或功能调整适时更新本隐私政策。更新后的政策将在本页面发布并注明更新日期。重大变更将通过扩展更新说明或 GitHub 仓库通知。
