# Security Policy

## Reporting a Vulnerability

If you believe you have found a security or privacy issue in WillowTab, please email `un1works@outlook.com`.

Please include:

- A short description of the issue.
- Steps to reproduce it.
- The affected browser and WillowTab version.
- Any relevant screenshots or console errors, with sensitive data removed.

Do not open a public GitHub issue for vulnerabilities or reports that include private data. This project does not currently operate a bug bounty program or publish a fixed response SLA.
