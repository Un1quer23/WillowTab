# WillowTab Improvement Ranking

## MUST-HAVE (Ranked by Impact)

| # | Improvement | Rationale | Confidence |
|---|------------|-----------|------------|
| 1 | **Performance: sub-100ms load time** | Minimal extensions (4.6-4.9★) outperform bloated ones (3.5★). Momentum's 17.5MB drives complaints. Sub-100ms is table stakes. | HIGH |
| 2 | **Daily wallpaper rotation as default** | Daily wallpaper novelty is the strongest retention mechanic — Momentum's #1 cited feature in 5-star reviews. 40-60% higher 7-day retention. | HIGH |
| 3 | **Zero/minimum permissions (MV3 trust signal)** | Extensions with minimal permissions have 20-30% higher install-to-active conversion. Momentum loses goodwill over invasive behavior. | HIGH |
| 4 | **Prominent "Rate us" feedback loop** | CWS ranking = ratings + downloads-vs-uninstalls. Smart review prompts see 3-5x more reviews than passive approaches. | HIGH |
| 5 | **Regional search engine support (Baidu focus)** | No competitor serves international users with regional engines. Bonjourr broken in China. WillowTab's Baidu support is unique. | HIGH |
| 6 | **Sync failure prevention (local-first reliability)** | Infinity users report data loss from sync failures, a top complaint. Local-first architecture avoids this trust-destroying failure mode. | HIGH |
| 7 | **Dark/light theme toggle** | Theme support is baseline expectation. "Can you add dark mode?" still common request for popular extensions. | MEDIUM-HIGH |
| 8 | **Clean onboarding (no promotional pop-ups)** | Momentum's 2M-user base eroding due to promotional interruptions. Zero-friction first-launch exploits competitor weakness. | HIGH |

| Improvement | Rationale | Confidence |

---

**Key Insight**: MUST-HAVE clusters around **performance/trust** (items 1, 3, 6, 8) and **engagement/growth** (items 2, 4, 5, 7). These exploit weaknesses in Momentum, Infinity, and Bonjourr while leveraging WillowTab's strengths (MV3, Baidu, local-first, zero ads).
